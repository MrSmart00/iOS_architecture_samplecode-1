# Fluxサンプル

![screen shot]()

Flux構成でボタンをタップすると背景色が変化するサンプルアプリを作成しました。

# 概要

* Change Colorで背景色をランダムで変更
* ランダムの結果、現在表示している色と同じだったらもう一度やり直し、必ず他の色が表示されるようにする
